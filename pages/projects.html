The Web4 Portfolio is a cutting-edge, Godot-based WebAssembly project designed to deliver an immersive web experience inspired by the cyberpunk genre. This portfolio showcases various web technologies and 3D elements, creating an interactive and visually engaging user interface. Developed using Godot 4.3, the project leverages the powerful capabilities of the Godot engine for both development and deployment. The portfolio includes custom 3D models, interactive elements, and a retro aesthetic achieved through assets like jdan/98.css and a modified retro hand cursor. The project is built for web distribution, ensuring compatibility and performance across modern web browsers.

Illegal Astronauts: Space-A-Gades

Novel Prose Science Fiction Marshall B McCloude

Illegal Astronauts: Space-A-Gades is a science fiction novel set in a universe where Earth has been lost to history. After a devastating solar war and a mass exodus from the Sol System, humanity has formed two factions: the Syndicated Colonies (a corporate-controlled society) and the Outcast Clans (a nomadic group of spacefaring rebels). Earth is forgotten and humanity now lives in various solar systems. Our most valuable resources are the Crux anomalies: rare materials essential for faster-than-light travel.
